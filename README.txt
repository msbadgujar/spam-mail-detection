
Spam Mail Detection System - Beginner Project (Python + Streamlit)

How to Run:

1. Install Python (3.8 or higher) from https://www.python.org/
2. Open VS Code and open this folder.
3. Open terminal and install required libraries:
   pip install streamlit scikit-learn pandas

4. In terminal, run:
   streamlit run app.py

5. A browser window will open with the Spam Detection App.

Files:
- app.py : Streamlit GUI App
- spam.csv : Sample dataset
- spam_classifier.pkl : Trained ML model
- vectorizer.pkl : Text vectorizer

No internet required to run the project once setup is done.
